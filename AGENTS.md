# Instructions pour l'Agent

- **Déploiement Cloud Run** : L'utilisateur préfère toujours déployer ses applications sur la région **Paris (`europe-west9`)**. Rappelez-lui de sélectionner cette région lors de ses déploiements pour s'assurer que les mappages de domaines personnalisés fonctionnent, car la région de Londres (`europe-west2`) ne les supporte pas.
