# Instructions pour l'Agent

- **Déploiement Cloud Run** : L'utilisateur préfère déployer ses applications sur les régions **Belgique (`europe-west1`)** ou **Pays-Bas (`europe-west4`)**. Rappelez-lui de sélectionner l'une de ces régions lors de ses déploiements pour s'assurer que les mappages de domaines personnalisés fonctionnent, car la région de Paris (`europe-west9`) ou Londres (`europe-west2`) ne le supportent pas toujours.
